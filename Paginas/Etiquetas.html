Con la etiqueta sup, el carácter seleccionado será un superíndice, p. ej. 1o. Este comando contiene el título del documento, que además se mostrará en la barra de título del buscador.
Con esta etiqueta HTML le podemos decir al navegador que viene un salto de línea. Nos sirve para hacer el texto más leíble.
